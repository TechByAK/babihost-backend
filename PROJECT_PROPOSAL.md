# Backend Developer Internship — Project Proposal

## Project Title

BabiHost Backend Booking API

## Project Description

This project is a backend API service for BabiHost, a hospitality platform that connects travelers with accommodation hosts.

The API will manage users, property listings, bookings, authentication, and basic CRUD operations.

## Main Functionalities

- User registration
- User login with token-based authentication
- Host property listing management
- Traveler booking requests
- Booking status management
- Input validation
- API documentation using Postman

## Technologies Used

- PHP
- Laravel
- MySQL
- Eloquent ORM
- Laravel Migrations
- Laravel Sanctum
- Postman
- GitHub

## Expected Outcome

The final result will be a Laravel backend API with database migrations, models, relationships, authentication, validation, and documented API endpoints.
